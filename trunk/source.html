Download

If you want to use the EidoGo SGF replayer on your own site, download the 'eidogo-player' zip from Google Code download page. See the included example.html file to get started.

If you end up using EidoGo on your site, !

Note that the JavaScript files contained in the 'eidogo-player' zip are compressed for speedier delivery over the web. For the full, uncompressed source, see below.

Also note that the version of the player on this site is using the most current, bleeding-edge code from the source code repository below, and not necessarily the latest milestone package linked above.

Source Code

EidoGo is open source and licensed under the BSD license. If you'd like to use the code under another license, such as the GPL, and I'll probably tell you to go right ahead.

If you want to get at the source code, you have a few options:

  1. Download the 'eidogo-source' zip from Google Code
  2. Browse it on the web
  3. Perform a checkout from the Subversion repository with the command: 
    svn co http://eidogo.googlecode.com/svn/trunk/ eidogo

Code contributions welcome!
